#  RetailBrain AI

## Smart Retail Shelf Monitoring System

RetailBrain AI is an AI-powered retail shelf monitoring system that detects products on supermarket shelves using Deep Learning object detection models.

---

## Features

- Product Detection
- Product Counting
- Shelf Occupancy Analysis
- Restocking Recommendation
- YOLOv8 Detection
- Faster R-CNN Comparison
- Streamlit Dashboard

---

## Models

- YOLOv8 Nano
- Faster R-CNN

---

## Dataset

SKU110K Dataset

---

## Technologies

- Python
- Streamlit
- Ultralytics YOLO
- PyTorch
- OpenCV
- NumPy
